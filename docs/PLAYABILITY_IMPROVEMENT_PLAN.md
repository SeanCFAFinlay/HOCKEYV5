# Playability Improvement Plan

## Completed This Pass

- Metadata-driven wave generation.
- Enemy role/tag unlock model.
- Endless and sandbox mode foundations.
- Touch placement preview before confirmation on phones.
- Mobile camera zoom/height target fix.

## Next Pass

## Playability Goals

- Early waves should teach tower roles without overwhelming the player.
- Boss waves should be readable and previewed.
- Mobile placement should never feel accidental.
- The bottom tower bar should remain useful without hiding required playfield rows.
